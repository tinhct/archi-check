# ArchiCheck

A CI/CD-integrated cognitive control plane that actively interrogates developer comprehension of AI-generated pull requests to prevent systemic deskilling.

## Overview

ArchiCheck is an open-source GitHub Application that gates pull requests based on the author's comprehension of the code. It analyzes Git diffs for complexity and AI reliance, generates interactive architectural quizzes in PR comments, and blocks merging until the human author successfully answers the quiz or an authorized lead triggers a break-glass override.
